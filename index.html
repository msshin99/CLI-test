2025 05 22 Git을 복습하는 내용압니다! main 브렌치 git init 맨 처음사용하는 명령어로 지금부터 이 폴더를 git으로 관리한다는 말이다. 한마디로 git이 나의 파일을 실시간으로 추적을 하는것이다. git status는 어떤 파일이 수정되었는지 알려주는것이다, git checkout -b NEW-BRANCH-NAME 브렌치를 새로 생성하는 이름이다 git branch는 생성된 브렌치가 어떤것인지 확인하는것이다 git merge 브렌치이름은 main브렌치와 병합을 하는것이다. history중에서 soft는 코드는 그대로 남겨지고 파일들을 삭제하는것이도 hard는 코드를 포함한 모든 파일을 삭제하는것이다.
